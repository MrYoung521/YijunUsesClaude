Lesson 4 · 第四课
Your First Project
你的第一个项目

🌟 Build a Book Post Generator

打造你的小红书绘本推文生成器

Your starter project: use Claude Code to build a simple tool that automatically writes a 小红书 post for any children's book you choose — in Chinese, with emojis, vocab tips, and hashtags included!

1

Install Claude Code

安装 Claude Code

Claude Code runs in a "terminal" — a text window on your computer. On a Mac, press Cmd + Space, type "Terminal", and press Enter. Then type the install command below and press Enter. Claude Code will set itself up automatically!

# Paste this into Terminal and press Enter:
npm install -g @anthropic-ai/claude-code
# 粘贴到Terminal后按回车即可安装
2

Start Claude Code

启动 Claude Code

In the same Terminal window, type claude and press Enter. You'll see a greeting — Claude Code is ready to help you! You can type in English or Chinese — it understands both perfectly.

claude
# 输入 claude 然后按回车启动
3

Give Your First Instruction

输入你的第一个指令

Now just type what you want — in plain language! Try copying this example and paste it into Claude Code. No coding knowledge needed — just describe what you want!

# Try typing this (in English or Chinese):
Create a Python script called book_post.py
that asks me for a children's book title,
then writes a complete 小红书 post in Chinese
with emoji, 3 English vocabulary tips,
and 8 relevant hashtags. Save the post to a file.

# 直接用中文说也可以！
4

Run Your New Tool!

运行你的新工具！

Claude Code will write and save the tool for you. Then you simply type the command below, enter a book name like "The Very Hungry Caterpillar", and watch it generate your post automatically — saved and ready to copy!

python3 book_post.py
# Enter any book title when asked
# 输入任何绘本书名即可生成推文
5

Copy, Paste & Post!

复制粘贴，发布！

Open the saved post file, copy the text, and paste it directly into 小红书. Add your photos and you're done! Once the tool is built, every future post takes you under 2 minutes to generate.

工具建好后，以后每篇推文2分钟搞定！🎉

💡

Yijun's Pro Tip: You can keep improving your tool by talking to Claude Code! Just say things like: "Make the posts shorter", "Add a section about age recommendations", or "Translate the hashtags to Chinese" — Claude Code will update your tool instantly.

一君小贴士：你可以不断跟 Claude Code 说话来改进你的工具！比如说"让文章短一点"、"加上适龄推荐"——它会立刻帮你更新！
